기능목록
1. 입력 Class
1) 경주 할 자동차 이름 저장 멤버
2) 시도할 회수 저장 멤버
2) 입력값 예외 처리 메소드
3) 경주 자동차 이름 입력 메소드
   문자열 입력받음. ‘,’로 파싱하여 저장
4) 시도 횟수 입력 메소드

2. 출력 Class
1) 각 차수별 실행 결과 출력 메소드
2) 최종 우승자 출력 메소드
   단독, 공동

3. 기능 Class
1)무작위 값 저장 멤버(해시 맵)
2)무작위 값 생성 메소드 : (0~9)까지 숫자
3)무작위 값 판단 메소드 : 4 이상일 때
